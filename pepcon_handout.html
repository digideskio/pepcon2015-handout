Sanders Kleinfeld

HTML5 Publishing Resources

HTMLBook Project (https://github.com/oreillymedia/HTMLBook)
Open Source resources for using HTML5 as a book authoring/publishing format: Schema, CSS stylesheets for print book design, and XSLT-based tools for producing ebook and Web outputs.
HTMLBook Specification (http://oreillymedia.github.io/HTMLBook/)
Latest Unofficial Working Draft specification and content model for HTMLBook.
W3C HTML5 Specification (http://www.w3.org/TR/html5/)
Official W3C specification for HTML5 and XHTML5.
HTML5 for Publishers (O'Reilly, 2011)
Free ebook by Sanders Kleinfeld on authoring next-generation interactive/multimedia ebook content in HTML5. Covers canvas, audio, video, MathML, and SVG

CSS Resources for print books

CSS Paged Media Level 3 (http://www.w3.org/TR/css3-page/)
W3C specification for CSS Paged Media, which provides a CSS styling/typesetting module geared toward PDF and print outputs
CSS Generated Content for Paged Media Module (http://dev.w3.org/csswg/css-gcpm/)
W3C specification for autogenerated text with Paged Media, for producing running headers/footers, cross-references, footnotes, and more using CSS.
Building Books with CSS3 (http://alistapart.com/article/building-books-with-css3)
Comprehensive introductory tutorial on developing print-book designs using CSS Paged Media by Nellie McKesson.
CSS for Print (screencast)
Four-part screencast series by Nellie McKesson that both covers the basics of CSS stylesheet development (from selectors to the box model), and provides a walkthrough of Paged Media's features for print compositing with CSS.

CSS print typesetting engines

Antenna House Formatter (http://www.antennahouse.com)
Commercial typesetting engine for producing professional print books using XHTML5 and CSS3. Supports CSS3 Paged Media, as well as some proprietary CSS extensions
Prince (http://www.princexml.com)
Commercial typesetting engine for producing professional print books using XHTML5 and CSS3. Supports CSS3 Paged Media; also supports DOM manipulation via embedded JavaScript in script elements.
Vivliostyle (https://github.com/vivliostyle/vivliostyle.js)
Currently in Alpha:Open source JavaScript engine for typesetting PDF in the browser. Limited CSS Paged Media support at this time. Also supports typesetting via EPUB Adaptive Layout (http://www.idpf.org/epub/pgt/).
